# Trained Models

Place exported Unity ML-Agents ONNX models here. The runtime accepts the release_21/Sentis `ModelAsset` and the 2.x/Barracuda `NNModel`.

The default scene configuration expects:

- `TankBattlePPO.onnx`

Keep `TankBattlePPO.onnx.meta` with the model so existing scene references keep the same Unity GUID.
